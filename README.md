# Каталог продуктового магазина на реакте
<a href="https://grzdrz.github.io/ProductSupermarket/"><p>Github Pages</p></a>

# Используемые библиотеки

* react
* redux
* react-router

# Загрузка
```
$ git clone https://github.com/grzdrz/ProductSupermarket.git
```

# Установка
```
$ npm install
```

# Сборка
```
$ npm run dev
$ npm run prod
```

# Запуск
```
$ npm run server
```

# Тестирование
```
$ npm run test
$ npm run test_debug
$ npm run coverage
```
